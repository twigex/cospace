# Support

Thanks for using Cospace! Here's where to get help, depending on what you need.

## 📖 Documentation

- **Manual & product docs:** [cospace.mytwigex.com](https://cospace.mytwigex.com)
- **Installation guide** (Helm chart, all MariaDB modes, ingress/TLS, LiveKit,
  EuroOffice/Collabora): [cospace.mytwigex.com/en/Installation](https://cospace.mytwigex.com/en/Installation)
  or [INSTALLATION.md](./INSTALLATION.md) in this repo
- **AI coding agents / Claude Code:** see [AGENTS.md](./AGENTS.md) and
  [SKILL.md](./SKILL.md) for agent-facing install/upgrade/troubleshoot procedures

## 🧪 Try it first

No install needed — the live demo is the same version you'd self-host:
[cospacedemo.twigex.com](https://cospacedemo.twigex.com)
(username: `cospace`, password: `cospace` — resets every 30 minutes, don't post
personal data)

## 💬 Community support

- **Community forum (Cospace family):** [discourse.twigex.com](https://discourse.twigex.com) —
  the primary place for installation help, bug reports, feature requests, and
  developer/API discussion. See the [Installation](https://discourse.twigex.com/c/installation/15)
  and [Bugs](https://discourse.twigex.com/c/bugs/9) categories specifically.
- **Reddit:** [r/cospace](https://www.reddit.com/r/cospace/) — general
  discussion and feedback
- **GitHub Issues:** for bugs or feature requests specific to the **Helm
  chart** in this repo, open a [GitHub Issue](https://github.com/twigex/cospace/issues)
- **Email:** [cospacesupport@twigex.com](mailto:cospacesupport@twigex.com)

## 🔒 Reporting a security issue

Please don't use these channels for security vulnerabilities — see
[SECURITY.md](./SECURITY.md) instead.

## 🏢 Commercial support / Organization Edition

For on-prem, hybrid, or cloud deployments with support, customization, and
complete data isolation, see [twigex.com](https://twigex.com) or get in touch
via the contact form there.
